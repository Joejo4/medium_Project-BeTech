# Medium Landing Page

A clean, responsive landing page clone inspired by [Medium](https://medium.com), built with plain HTML and CSS.

## Project Structure

```
medium-landing-page/
├── index.html
├── styles.css
├── medium.png
└── README.md
```

## Features

- Responsive navigation with a hamburger menu for mobile screens
- Hero section with a headline, tagline, and call-to-action button
- Features section highlighting the platform's value proposition
- Clean footer
- Pure HTML & CSS — no frameworks or JavaScript

## Getting Started

1. Clone or download the project files.
2. Open `index.html` in your browser.

No build tools or dependencies required.

## Responsive Design

The layout adapts to screen sizes below **768px**:
- The navigation links collapse into a hamburger menu (CSS-only toggle using a hidden checkbox)
- Hero heading font size scales down for readability

## Customization

- **Colors & Fonts** — Edit `styles.css` to update the color scheme or typography.
- **Content** — Update text and links directly in `index.html`.
- **Logo/Favicon** — Replace `medium.png` with your own image.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
